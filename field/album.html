User-Generated Album Art

For the holiday season, I wanted to create something I could potentially give to my friends and family — something both personal and useful. I’m big on connecting with others through music, so I created an album cover generator in Processing to use for my next batch of mixes.

→ Download the ZIP file to create your own album art! Requires Processing application.

This project relies on user input – the name and a descriptor of the recipient – to produce a visual output – the personalized visual. Below is the journey the user takes through the application, as well as examples of the visual output.
